# Student Placement Dashboard

## Overview

Created an Excel dashboard to analyze student placement trends, employability skills, and future career readiness through interactive visualizations and reports.

## Tools Used

* Microsoft Excel

## Features

* Interactive dashboard
* Placement overview analysis
* Skills and employment analysis
* AI readiness and future trends insights
* KPI cards and charts

## Skills Demonstrated

* Data Cleaning
* Data Analysis
* Pivot Tables
* Charts and Visualizations
* Dashboard Design
